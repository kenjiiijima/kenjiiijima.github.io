このサイトは、GitHub Pages を使って公開されています。  
特別な技術者向けのサイトではありません。

## 無料であることについて

GitHub Pages は、個人利用であれば無料で使えます。  
広告は表示されません。  
閲覧者に登録を求めることもありません。

この点は、このサイトの前提条件でした。

## なぜ GitHub なのか

理由は単純です。

- 文章や記録を、あとから修正できる  
- 変更の履歴が残る  
- 必要なら、過去の状態に戻れる  
- 公開するものと、非公開のものを分けて管理できる  

これは「便利だから」というよりも、  
**考えながら作るための条件**でした。

## 本や記事との違い

このサイトにある文章は、  
最初から完成形であることを前提にしていません。

途中で言い直されたり、  
構成が変わったりすることがあります。

GitHub は、そうした変化を  
無理に隠さず、そのまま残すことができます。

## 向いている人・向いていない人

このやり方は、すべての人に向いているわけではありません。

- 自分で考えながら作りたい人  
- 後戻りしながら進むことを自然だと思える人  

には向いています。

一方で、

- 決まった正解が欲しい人  
- 最短手順だけを知りたい人  

には、別の方法の方が合っているかもしれません。

## 最後に

もしあなたが、

- 自分の文章や考えを置く場所を探していて  
- 広告や余計な装飾を避けたいと思い  
- 途中経過も含めて残したい  

と感じているなら、  
このサイトの作り方は、ひとつの参考になるかもしれません。

それ以上の主張は、ここにはありません。
