# GitHub Pages でホームページを作る（これだけ見れば動く）

このページは、  
**GitHub Pages を使って、実際にホームページを立ち上げるための記録**です。

途中で止まらず、  
このページだけを上から順に見れば、  
最低限の形でホームページが立ち上がります。

---

## 0. いまやること・やらないこと

### やること
- 手順どおりに進む
- 分からなくても先に進む

### やらないこと
- 完璧に理解しようとしない
- デザインを考えない
- 公開・非公開を気にしない
  
---

## 1. GitHub にサインアップする

1. https://github.com にアクセス
2. **Sign up** を選ぶ
3. ユーザー名を決める

### 注意
- ユーザー名は URL に使われます
- この時点で完璧である必要はありません
- 後から変更できます（URLは変わります）
---

## 2. Pages 用リポジトリを作る

ログイン後：

1. 右上の「＋」→ **New repository**
2. **Repository name** に次を入力

3. 設定は以下のみ
- Public
- **Add README.md：ON**

4. **Create repository** を押す
---

## 3. README.md を編集する（トップページ）

README.md の内容が、  
**そのままホームページのトップページ**になります。

1. README.md を開く
2. 鉛筆マーク（Edit）を押す
3. 次のように書く（内容は仮でよい）

```markdown
# サイト名

このサイトについての簡単な説明
---

---

続けてよろしければ、  
**「次へ」** とだけ言ってください。
