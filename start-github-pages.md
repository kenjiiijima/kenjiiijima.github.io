# GitHub Pages でホームページを作る（最短）

このページは、GitHub Pages を使ってホームページを立ち上げるための
**最短手順の記録**です。説明は最小限にしています。

---

## 1. GitHub アカウントを作る

ユーザー名は URL に使われます。
この時点で完璧である必要はありません。

---

## 2. Pages 用リポジトリを作る

リポジトリ名は、次の形式にします。

README.md を追加した状態で作成します。

---

## 3. README.md を編集する

README.md の内容が、そのままトップページになります。
保存（Commit）して、少し待ちます。

---

## 4. 表示を確認する

反映されない場合は、Ctrl + F5 を試してください。
