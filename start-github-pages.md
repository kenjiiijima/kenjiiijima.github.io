# GitHub Pages でホームページを作る  
――これだけ見れば、実際に立ち上がる

この文章は、  
GitHub Pages を使って、実際にホームページを立ち上げるための作業記録である。

このページを上から順に読み、  
同じ操作を行えば、最低限の形でホームページが立ち上がる。

---

### 本資料の位置づけについて

本資料は、  
GitHub Pages を使ってホームページを立ち上げるための  
個人的な作業記録を公開したものである。

営利目的ではなく、  
特定のサービスや商品を勧誘・販売する意図はない。

## Step 1　GitHub にサインアップする

まず、GitHub のアカウントを作る。

1. ブラウザで https://github.com にアクセスする  
2. 画面右上の「Sign up」を選ぶ  
3. ユーザー名を決める  

ユーザー名は、  
あとで作るホームページの URL に使われる。

この時点で完璧である必要はない。

---

## Step 2　Pages 用リポジトリを作る

ログイン後、ホームページ本体を置く場所を作る。

1. 画面右上の「＋」を押し、「New repository」を選ぶ  
2. Repository name に、次の形式で名前を入力する  

　　ユーザー名.github.io  
　　例えば、tarou.github.io  

3. 設定は、次の2点だけ確認する  

　　Public  
　　Add README.md を ON  

4. 「Create repository」を押す  

---

## Step 3　README.md を編集する（トップページ）

README.md は、そのままホームページのトップページになる。

1. README.md を開く  
2. 鉛筆マーク（Edit）を押す  

ここで、次の文章を書いてみる。

　　太郎のホームページ  
　　わたくし太郎は、ここに研究記録を書いてゆきます。

3. 「Commit changes」を押す  

---

## Step 4　表示を確認する

ブラウザで、次の URL を開く。

　　https://ユーザー名.github.io  

反映されない場合は、  
Ctrl + F5（強制更新）を試す。

---

## Step 5　タイトルが自動表示される場合

意図していないタイトルが表示される場合は、  
設定ファイルを追加する。

1. 新しいファイルを作成する  
2. ファイル名を次のようにする  

　　_config.yml  

3. 中身を次の2行だけにする  

　　title:  
　　description:  

4. 保存する  

---

## Step 6　ここまでで出来ていること

この時点で、  
GitHub Pages によるホームページは立ち上がっている。

・無料  
・広告なし  
・自分専用の URL  
・あとから修正できる
