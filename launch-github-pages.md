# GitHub Pages を実際に立ち上げる（これだけ見れば動く）

このページは、
「GitHub を使ってみようと思った人が、
途中で止まらずにホームページを立ち上げる」
ための **実作業用の手順**です。

比較や評価はしません。
選択肢も増やしません。
この順にやれば、立ち上がります。

---

## 0. いまやること・やらないこと

**やること**
- 手順どおりに進む
- 分からなくても先に進む

**やらないこと**
- 完璧に理解しようとしない
- デザインを考えない
- 公開・非公開を気にしない

---

## 1. GitHub にサインアップする

1. https://github.com にアクセス
2. Sign up を選ぶ
3. ユーザー名を決める

### ここでの注意
- ユーザー名は URL に使われます
- **仮でかまいません**
- 後から変更できます（URLは変わります）

---

## 2. Pages 用リポジトリを作る

ログイン後：

1. 右上の「＋」→ **New repository**
2. Repository name に次を入力

3. 設定は以下だけ
- Public
- **Add README.md：ON**

4. Create repository を押す

---

## 3. README.md を編集する（トップページ）

README.md は、
**そのままホームページの内容**になります。

1. README.md を開く
2. 鉛筆マーク（Edit）を押す
3. いまは、これだけ書く

```markdown
# サイト名

ここに文章を書いていきます。
