# 本当に使えるGitHubの立ち上げ方

この記事では、GitHubを使って  
広告なし・無料でホームページを立ち上げる手順を、  
実際につまずいた点を含めて記録します。

## GitHubを必要とするのはこういう人
- 広告なしで、長く使える公開場所がほしい
- 記事や翻訳を、更新しながら蓄積したい
- 公開／非公開を分けて管理したい

## GitHubは何が優れているか
- 無料でも広告が出ない
- 変更履歴（版）が自動で残る
- 公開サイトと非公開保管庫を同時に持てる

## 実際につまずいたところ
- サインアップ時の人間確認（パズル）
- ユーザー名はどこまで公開されるのか
- READMEとは何か
- 更新したのに表示が変わらない（キャッシュ）

これらはすべて、初めてだと自然につまずく点でした。
