# このサイトの構成について

このサイトは、  
哲学・科学技術・AI に関する制作記録と、  
公開できる成果物を置くための場所である。

内容は、完成したものだけでなく、  
作成途中の記録も含んでいる。

---

## 記事・記録について

各ページは、  
特定のテーマや作業単位ごとに分けて記載している。

完成度や順序は固定していない。  
必要に応じて、追記・修正される。

---

### GitHub について

本サイトの一部では、  
記録や公開の手段として GitHub Pages を利用している。

これは、  
作業手順を簡潔に残すための選択であり、  
特定のサービスを推奨することを目的としたものではない。

GitHub Pages を使った立ち上げ方法については、  
必要に応じて次のページを参照されたい。

→ [GitHub Pages でホームページを作る](start-github-pages.md)
